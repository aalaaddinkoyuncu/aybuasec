# ASEC Website (aybuasec)

Aybü ASEC (Anadolu Üniversitesi Siber Güvenlik Kulübü) için geliştirilmiş PHP tabanlı web sitesi.

## Özellikler

- Blog ve haber yönetimi
- Etkinlik ve duyuru sistemi
- Galeri
- Admin paneli
- PDO ile güvenli veritabanı bağlantısı

## Teknolojiler

- PHP
- MySQL
- CSS / SCSS
- Composer

## Kurulum

```bash
composer install
```

Veritabanını `db_asec.sql` dosyasıyla oluştur, ardından `db.php` içindeki bağlantı bilgilerini güncelle.
